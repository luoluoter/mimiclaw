当前浏览器不支持 Web Serial，请换 Chrome / Edge。 这个页面需要在安全上下文中打开，建议使用 http://127.0.0.1:8787/flasher。 
./scripts/build_merged_bin.sh

使用提示

  • 页面地址建议直接打开 http://127.0.0.1:8787/flasher。
  • 首次点击按钮时，浏览器会弹串口选择框，你要选设备对应的 USB 串口。
  • 如果板子有两个口，优先选原生 USB/JTAG 口。
  • 烧录失败时，先断开其它串口程序，再重试。
  • 这个页面烧的是整包镜像，适合首刷和分发，不是 OTA 升级页。
构建后端服务:
cd bridge_service
MIMI_BRIDGE_SKIP_FRPC=1 ./start.sh

打开页面:
http://127.0.0.1:8787/flasher
